
Test small implementation of an JWT token signin in Spring-Boot-3.2.1 and spring-Security-6

There is no UI here

To Generate a JSON Web Token - Create a POST request with url localhost:8080/authenticate with body
{
   "username":"admin",
   "password":"test"
}
